\section{实验目标}
基于 baseline CNN（至少3个卷积层和1个池化层），在 STL10 数据集上开展优化实验，重点比较：
\begin{itemize}
  \item 数据增强是否提升泛化能力；
  \item BatchNorm 与 Dropout 对收敛和过拟合的影响；
  \item 优化器与学习率设置对最终测试表现的影响。
\end{itemize}

\section{实验设置}
\begin{itemize}
  \item 数据来源：\texttt{STL10/train} 与 \texttt{STL10/test}。
  \item 训练配置：固定 \texttt{epochs=50}、\texttt{batch\_size=128}、\texttt{val\_ratio=0.1}、\texttt{seed=42}。
  \item 评估指标：Accuracy、Macro-F1、Weighted-F1、分类报告、混淆矩阵。
  \item 输出目录：\texttt{outputs\_improved/}。
\end{itemize}

\section{实验分组}
\begin{itemize}
  \item baseline：无增强、无 BN、无 Dropout（Adam, lr=0.001）
  \item exp1\_aug：数据增强（RandomCrop + HorizontalFlip + ColorJitter）
  \item exp2\_aug\_bn：数据增强 + BatchNorm
  \item exp3\_aug\_bn\_dropout：数据增强 + BatchNorm + Dropout(0.3)
  \item exp4\_opt\_adam / exp4\_opt\_sgd：在实验三基础上对比优化器与学习率
  \item exp5\_best\_combo：综合优化（AdamW, lr=0.0008）
\end{itemize}

\section{结果分析}
\subsection{1) 有效实验与主要收益来源}
从总体结果看，本实验中最有效的两个改进是\textbf{数据增强}和\textbf{Batch Normalization}。baseline 的测试准确率为 54.70\%，加入随机裁剪、水平翻转和颜色扰动后，\texttt{exp1\_aug} 提升到 66.50\%，绝对提升 11.80 个百分点，说明原始 baseline 的主要问题是泛化能力不足。进一步加入 BatchNorm 后，\texttt{exp2\_aug\_bn} 达到最高测试准确率 71.70\%，相比 baseline 绝对提升 17.00 个百分点，是本组实验中最有价值的配置。

BatchNorm 的收益主要体现在两方面：一是稳定中间特征分布，使训练过程更平稳；二是在数据增强已经扩大样本扰动范围后，帮助模型更好地适应不同输入分布。因此，\texttt{exp2\_aug\_bn} 同时取得最高 Best Val Acc、Test Acc、Macro-F1 和 Weighted-F1，可作为最终推荐模型。

\subsection{2) 收益有限或需要重新调参的实验}
\texttt{exp3\_aug\_bn\_dropout} 在 \texttt{exp2} 的基础上加入 Dropout(0.3)，测试准确率为 71.00\%，略低于 \texttt{exp2\_aug\_bn}。这说明 Dropout 确实有抑制过拟合的作用，但当前 0.3 的强度并没有带来额外收益，可能已经削弱了特征表达能力。后续若继续使用 Dropout，应尝试 0.1 或 0.2，并与学习率重新配合调参。

\texttt{exp4\_opt\_adam} 与 \texttt{exp4\_opt\_sgd} 用于比较优化器。结果显示 Adam 分支为 70.50\%，SGD 分支为 71.60\%，二者差距不大，SGD 略优但没有超过 \texttt{exp2\_aug\_bn}。因此优化器调整属于细节优化，收益小于数据增强和 BatchNorm。 \texttt{exp5\_best\_combo} 使用 AdamW 和更小学习率，测试准确率为 70.70\%，也未超过 \texttt{exp2}，说明简单叠加正则化和优化器改动并不必然提升效果。

\subsection{3) 过拟合分析}
baseline 的 train\_acc 接近 100\%，而 val\_acc 和 test\_acc 明显偏低，属于典型过拟合。数据增强能显著缓解这一问题，因为它让模型在训练阶段看到更多位置、颜色和局部形变变化；BatchNorm 进一步提高了训练稳定性和泛化上限。相比之下，Dropout 和残差连接虽然理论上有助于泛化和深层训练，但在当前任务和超参数下并不是主要收益来源。

\subsection{4) 最终测试集性能}
最优实验为 \texttt{exp2\_aug\_bn}：
\begin{itemize}
  \item Test Accuracy = 71.70\%
  \item Macro-F1 = 0.7178
  \item Weighted-F1 = 0.7178
\end{itemize}
相比 baseline（54.70\%），绝对提升约 17 个百分点，说明“数据增强 + BatchNorm”是最有效的优化策略。

\subsection{5) 方法优劣总结}
\begin{itemize}
  \item 数据增强：从 54.70\% 提升到 66.50\%，是最关键的泛化改进。
  \item BatchNorm：在数据增强基础上进一步提升到 71.70\%，是最终最优组合。
  \item Dropout：当前 0.3 设置收益有限，可能偏强，需要重新调参。
  \item 优化器对比：SGD(0.01) 与 Adam(0.001)接近，SGD 分支略优，但不是决定性因素。
\end{itemize}

\section{可解释性分析（Grad-CAM）}
\subsection{方法说明}
Grad-CAM 用目标类别在 logits 上的标量得分对\textbf{最后一层卷积特征图}求梯度，把梯度在
空间上做一次全局平均得到每个通道的权重，再对特征图通道加权求和并取 ReLU，得到一张与
卷积特征同分辨率的类激活图，最后上采样到输入尺寸并伪彩色显示。直观上，它回答的是：
模型为了把样本判成某一类，更依赖特征图里哪些空间位置（对应到原图大致落在哪些区域）。

本实验在最后一层 \texttt{Conv2d} 上挂钩实现（未改动网络结构），热力图对应当前预测类别
的解释（即对 $\arg\max$ 类别的 Grad-CAM）。脚本使用与测试阶段一致的预处理（Resize/CenterCrop、
Normalize(0.5)）。

\subsection{可视化设置}
模型选用测试集表现最好的 \texttt{exp2\_aug\_bn} 分支对应的 checkpoint；从 STL-10 官方
划分测试集中筛选样本，分别给出\textbf{预测正确}与\textbf{预测错误}两类可视化，结果保存
在 \texttt{homework/figures/gradcam/}（含 \texttt{correct/}、\texttt{wrong/} 子目录）。
本地复现可运行：
\begin{verbatim}
python src/run_gradcam.py --checkpoint outputs_improved/exp2_aug_bn/checkpoints/exp2_aug_bn_best.pth
\end{verbatim}

\subsection{分析与讨论}
STL-10 图像主体相对居中、背景相对干净，若模型学到的是可迁移的语义线索，热力图应主要覆盖
物体部位而不是大片纯色背景。实验观察大致符合这一点：在预测正确的样本上，热点往往贴在物体
外轮廓与内部纹理较强区域；这与人类判断 ``物体在哪'' 的经验较为一致。

在预测错误的样本上，更容易看到两类现象：一是响应分散或偏向边缘/角落，说明模型可能被局部
噪声带走；二是热点落在与标签类别相近的部位但在细粒度上混淆（例如动物类之间），这时热力图
不一定 ``乱''，但它强调的语义区域与真实标签不一致，提示模型仍在做 ``看起来像'' 的关联，
而非完全稳健的高层语义。

\subsection{小结}
综合来看，模型并非只在像素层面做模板匹配：正确样本的可视化具有一定语义指向性；但错误样本
说明决策仍会被背景或细粒度混淆干扰，距离 ``完全可靠的高层语义理解'' 仍有差距。这与测试集
准确率约 72\%（相对 baseline 明显提升但仍有限）相吻合。

\section{结论}
本实验完成了 STL10 图像分类任务中的 baseline 与多组优化对比。结果表明：在当前数据与配置下，\textbf{数据增强 + BatchNorm} 是最优组合，测试准确率达到 71.70\%，显著优于 baseline 的 54.70\%。数据增强主要改善泛化能力，BatchNorm 进一步稳定训练并提高性能上限；Dropout、AdamW 和优化器替换在当前设置下没有超过该组合。

Grad-CAM 可视化进一步表明模型在多数正确样本上会关注物体区域，但错误样本仍存在背景干扰与细粒度混淆，说明模型已经学到一定语义线索，但距离完全稳健的图像理解仍有改进空间。后续可继续尝试更精细的数据增强策略、学习率调度搜索和更深层网络结构，并结合更强的可解释性方法做对照。
