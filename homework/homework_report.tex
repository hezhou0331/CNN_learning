\section{实验目标}
基于 baseline CNN，在 STL10 数据集上开展优化实验，尝试在模型训练过程当中可能会影响模型效果的因素，更好地掌握上课学习到的内容
重点比较不同因素对模型效果的影响，包括：
\begin{itemize}
  \item 数据增强是否提升泛化能力；
  \item BatchNorm 与 Dropout 对收敛和过拟合的影响；
  \item 优化器与学习率设置对最终测试表现的影响；
  \item 更深网络（\texttt{improved\_long}，多段池化）相对浅层 configurable 的收益。
\end{itemize}
同时还通过Grad-CAM进行可解释性的分析，更好地理解模型为什么能够做出这样的预测

\section{实验设置}
\begin{itemize}
  \item 数据来源：\texttt{STL10/train} 与 \texttt{STL10/test}。（课程提供）
  \item 训练配置：\texttt{batch\_size=128}、\texttt{val\_ratio=0.1}、\texttt{seed=42}。训练集按固定随机种子划分出验证集，验证集用于选择最佳 checkpoint；如果验证集准确率连续 \texttt{patience=20} 个 epoch 没有刷新最优值，则提前停止训练。
  \item 评估指标：Accuracy、Macro-F1、Weighted-F1、分类报告、混淆矩阵。
  \item 输出目录：\texttt{outputs\_improved/}。（可在压缩包或github上查看）
\end{itemize}


\section{实验分组与网络结构说明}
实验在 baseline 上共 9 组改进设置，覆盖数据增强、BatchNorm、Dropout、优化器、学习率、网络深度，以及在同配方下增加卷积层数等。
\begin{itemize}
  \item baseline：基础 CNN，无增强、无 BN、无 Dropout（Adam, lr=0.001, L2 \texttt{weight\_decay=5e-4}）
  \item exp1\_aug：数据增强（RandomCrop + HorizontalFlip + ColorJitter）
  \item exp2\_aug\_bn：数据增强 + BatchNorm
  \item exp3\_aug\_dropout：数据增强 + Dropout(0.3)
  \item exp4\_aug\_bn\_sgd：与 exp2 相同（增强 + BN、无 Dropout），优化器改为 SGD（\texttt{lr=0.01}）
  \item exp5\_aug\_bn\_lr：与 exp2 相同，Adam 学习率改为 \texttt{3e-4}
  \item exp6\_improved\_long\_6：与 exp2 相同，卷积层数改为6
  \item exp7\_improved\_long\_9：与 exp2 相同，卷积层数改为9
  \item exp8\_improved\_longer\_12：与 exp2 相同，卷积层数改为12
  \item exp9\_aug\_bn\_15conv：与 exp2 相同，卷积层数改为15
\end{itemize}

\section{可解释性分析（Grad-CAM）}
\subsection{方法说明}
Grad-CAM 的作用是看模型做判断时主要关注了图片里的哪些位置。它会利用最后一层卷积特征图
和类别得分的梯度，算出不同通道的重要程度，再合成一张热力图。热力图越亮的地方，说明模型
在预测这个类别时越依赖这些区域。

本实验没有改动网络结构，只是在最后一层 \texttt{Conv2d} 上取特征和梯度。这里解释的是模型
当前预测出来的类别，也就是 $\arg\max$ 对应的类别。输入图片的预处理方式和测试阶段保持一致，
包括 Resize、CenterCrop 和 Normalize(0.5)。

\subsection{分析与讨论}
模型选用当前测试集表现最好的 \texttt{exp6\_improved\_long\_6} 对应的 checkpoint。样本来自 STL-10
官方测试集；为便于对比，从测试集中分别选取 \textbf{2} 个预测正确与 \textbf{4} 个预测错误的样本
做 Grad-CAM。每张导出图均为「原图—热力图—叠加」三列排版。下文选取 2 张正确样本和
4 张错误样本作为主图展示；脚本也会把更多结果分别保存到 correct 与 wrong 子目录，便于复查。

\section{结论}
本次课程项目完成了 STL10 图像分类任务中的 baseline 与多组改进实验。整体结果来看，数据增强和 BatchNorm
对浅层模型提升比较明显；继续加深网络后，\texttt{exp6\_improved\_long\_6} 取得了本实验中最高的测试准确率
\textbf{81.80\%}。同时，\texttt{exp9\_aug\_bn\_15conv} 的结果也让我看到，单纯增加卷积层数不一定就会更好，
模型结构、池化方式、训练轮数和正则化都需要一起考虑。

这次项目对我来说最大的收获不是只跑出了一个最高准确率，而是比较完整地走了一遍深度学习实验流程。
五一期间我集中学习了 PyTorch，从一开始看不太懂 \texttt{Dataset}、\texttt{DataLoader}、\texttt{forward}
和训练循环，到后面能够自己修改模型结构、调整实验配置、重新训练、整理曲线和分析结果，这个过程让我
对卷积神经网络的训练有了更具体的认识。Grad-CAM 的可视化也让我意识到，模型的准确率只是一个方面，
还需要看它到底关注了图像里的哪些区域，才能更好地理解模型为什么会预测正确或出错。
