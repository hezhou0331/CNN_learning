\section{实验目标}
基于 baseline CNN，在 STL10 数据集上开展优化实验，尝试在模型训练过程当中可能会影响模型效果的因素，更好地掌握上课学习到的内容
重点比较不同因素对模型效果的影响，包括：
\begin{itemize}
  \item 数据增强是否提升泛化能力；
  \item BatchNorm 与 Dropout 对收敛和过拟合的影响；
  \item 优化器与学习率设置对最终测试表现的影响；
  \item 更深网络（\texttt{improved\_long}，多段池化）相对浅层 configurable 的收益。
\end{itemize}
同时还通过Grad-CAM进行可解释性的分析，更好地理解模型为什么能够做出这样的预测

\section{实验设置}
\begin{itemize}
  \item 数据来源：\texttt{STL10/train} 与 \texttt{STL10/test}。（课程提供）
  \item 训练配置：固定 \texttt{epochs=50}、\texttt{batch\_size=128}、\texttt{val\_ratio=0.1}、\texttt{seed=42}。
  \item 评估指标：Accuracy、Macro-F1、Weighted-F1、分类报告、混淆矩阵。
  \item 输出目录：\texttt{outputs\_improved/}。（可在压缩包或github上查看）
\end{itemize}

\section{实验分组}
\begin{itemize}
  \item baseline：基础 CNN，无增强、无 BN、无 Dropout（Adam, lr=0.001, L2 \texttt{weight\_decay=5e-4}）
  \item exp1\_aug：数据增强（RandomCrop + HorizontalFlip + ColorJitter）
  \item exp2\_aug\_bn：数据增强 + BatchNorm
  \item exp3\_aug\_dropout：数据增强 + Dropout(0.3)
  \item exp4\_aug\_bn\_sgd：与 exp2 相同（增强 + BN、无 Dropout），优化器改为 SGD（\texttt{lr=0.01}）
  \item exp5\_aug\_bn\_lr：与 exp2 相同，Adam 学习率改为 \texttt{3e-4}
  \item exp6\_improved\_long：数据增强 + \texttt{improved\_long}（6 个卷积层 + 六段空间下采样 + 分类头 Dropout），Adam \texttt{lr=1e-3}
  \item exp7\_improved\_longer：与 exp6 相同训练配方，网络为 \texttt{improved\_longer}（\textbf{9} 个卷积层，无残差）
  \item exp8\_improved\_longer\_12：与 exp7 相同训练配方，网络为 \texttt{improved\_longer\_12}（\textbf{12} 个卷积层，在 \texttt{improved\_longer} 的 24$\times$24 与 12$\times$12 段各加深若干 128 通道卷积；无残差）
\end{itemize}

\section{结果分析}
\subsection{1) 有效实验与主要收益来源}
从总体结果看，\textbf{数据增强}与\textbf{Batch Normalization} 仍是将 baseline（54.70\%）抬升到可用水平的关键；\texttt{exp2\_aug\_bn} 在浅层 configurable 上达到 71.70\%。\texttt{exp6\_improved\_long}（6 个卷积层）将测试准确率提升到 80.40\%，验证峰值 82.71\%。进一步将骨干加深为 \texttt{improved\_longer}（\textbf{9} 个卷积层、无残差）后，\texttt{exp7\_improved\_longer} 在测试集上达到 \textbf{81.60\%}，为\textbf{当前最高测试准确率}；说明在固定训练配方下，\textbf{适度增加卷积深度}仍有边际收益。\texttt{exp8\_improved\_longer\_12} 在相同配方下将卷积层数增至 \textbf{12}，测试准确率降至 \textbf{69.50\%}、验证峰值约 70.29\%，训练在第 22 轮触发早停（最优验证在第 14 轮），表明在\textbf{不调整学习率、正则与训练轮次}的前提下，\textbf{继续堆叠卷积}反而损害泛化，可能因容量过大导致过拟合或优化 landscape 变差。

\texttt{exp1\_aug} 相对 baseline 提升 11.80 个百分点，说明泛化不足是原始模型的主要瓶颈；\texttt{exp3\_aug\_dropout}（69.00\%）表明在无 BN 时 Dropout 有帮助但仍难超过引入 BN 的浅层模型。

\subsection{2) 优化器与学习率（exp4 / exp5）}
\texttt{exp4\_aug\_bn\_sgd} 与 \texttt{exp2} 结构完全一致，仅将优化器换为 SGD（\texttt{lr=0.01}），测试 70.30\%，低于 \texttt{exp2} 的 71.70\%，说明在该小网络上 Adam 更匹配当前训练配方。\texttt{exp5\_aug\_bn\_lr} 将 Adam 学习率降到 \texttt{3e-4}，测试 69.80\%，进一步说明\textbf{盲目降 lr 不一定提升}，需要与调度及有效 batch 统计等联合考虑。

\subsection{3) 过拟合与模型深度}
baseline 的 train\_acc 接近 100\% 而 val/test 明显偏低，属典型过拟合。数据增强与 BN 可显著缓解。\texttt{exp6}、\texttt{exp7} 等深模型借助 BN、Dropout 与池化，测试表现整体较好；\texttt{exp7} 在测试上略优于 \texttt{exp6}，而 \texttt{exp6} 的验证峰值更高，提示二者泛化特性略有不同。\texttt{exp8} 在更深骨干上验证与测试均明显低于 \texttt{exp7}，与 ``深度—数据量—正则'' 不匹配时的退化现象一致。

\subsection{4) 最终测试集性能}
当前测试集最优为 \texttt{exp7\_improved\_longer}：
\begin{itemize}
  \item Test Accuracy = 81.60\%
  \item Macro-F1 = 0.8153
  \item Weighted-F1 = 0.8153
\end{itemize}
\texttt{exp6\_improved\_long} 测试 80.40\%、验证峰值更高（82.71\%）；浅层 \texttt{exp2\_aug\_bn} 为 71.70\%，可作轻量备选。\texttt{exp8\_improved\_longer\_12} 测试 69.50\%、Macro/Weighted-F1 约 0.6948，明显低于 \texttt{exp7}。

\subsection{5) 方法优劣总结}
\begin{itemize}
  \item L2 正则化：各组统一 \texttt{weight\_decay=5e-4}，不是组间差异的主要来源。
  \item 数据增强：相对 baseline 提升最大的一块基础能力。
  \item BatchNorm：与增强配合构成高质量浅层模型（\texttt{exp2}）。
  \item Dropout：在 \texttt{exp3} 中体现无 BN 时的正则价值；在深模型分类头中与 BN 配套。
  \item 更深网络：\texttt{improved\_long} $\rightarrow$ \texttt{improved\_longer} 持续提升测试精度；再增至 \texttt{improved\_longer\_12}（\texttt{exp8}）在本设置下\textbf{未}带来收益，说明深度需与数据规模及超参一并调节。
  \item 优化器/学习率：在固定 \texttt{exp2} 结构下，SGD 与过小 Adam lr 未超过原 \texttt{exp2}。
\end{itemize}

\section{可解释性分析（Grad-CAM）}
\subsection{方法说明}
Grad-CAM 用目标类别在 logits 上的标量得分对\textbf{最后一层卷积特征图}求梯度，把梯度在
空间上做一次全局平均得到每个通道的权重，再对特征图通道加权求和并取 ReLU，得到一张与
卷积特征同分辨率的类激活图，最后上采样到输入尺寸并伪彩色显示。直观上，它回答的是：
模型为了把样本判成某一类，更依赖特征图里哪些空间位置（对应到原图大致落在哪些区域）。

本实验在最后一层 \texttt{Conv2d} 上挂钩实现（未改动网络结构），热力图对应当前预测类别
的解释（即对 $\arg\max$ 类别的 Grad-CAM）。脚本使用与测试阶段一致的预处理（Resize/CenterCrop、
Normalize(0.5)）。

\subsection{可视化设置}
模型选用当前测试集表现最好的 \texttt{exp7\_improved\_longer} 分支对应的 checkpoint；从 STL-10 官方
划分测试集中筛选样本，分别给出\textbf{预测正确}与\textbf{预测错误}两类可视化，结果保存
在 \texttt{homework/figures/gradcam/}（含 \texttt{correct/}、\texttt{wrong/} 子目录）。
本地复现可运行：
\begin{verbatim}
python src/run_gradcam.py --checkpoint outputs_improved/exp7_improved_longer/checkpoints/exp7_improved_longer_best.pth
\end{verbatim}

\subsection{分析与讨论}
STL-10 图像主体相对居中、背景相对干净，若模型学到的是可迁移的语义线索，热力图应主要覆盖
物体部位而不是大片纯色背景。实验观察大致符合这一点：在预测正确的样本上，热点往往贴在物体
外轮廓与内部纹理较强区域；这与人类判断 ``物体在哪'' 的经验较为一致。

在预测错误的样本上，更容易看到两类现象：一是响应分散或偏向边缘/角落，说明模型可能被局部
噪声带走；二是热点落在与标签类别相近的部位但在细粒度上混淆（例如动物类之间），这时热力图
不一定 ``乱''，但它强调的语义区域与真实标签不一致，提示模型仍在做 ``看起来像'' 的关联，
而非完全稳健的高层语义。

\subsection{小结}
综合来看，模型并非只在像素层面做模板匹配：正确样本的可视化具有一定语义指向性；但错误样本
说明决策仍会被背景或细粒度混淆干扰，距离 ``完全可靠的高层语义理解'' 仍有差距。这与测试集
准确率约 81--82\%（\texttt{exp7} / \texttt{exp6}）相吻合。

\section{结论}
本实验完成了 STL10 图像分类任务中的 baseline 与多组优化对比。结果表明：\textbf{数据增强 + BatchNorm} 可将浅层模型推到 71.70\%；\texttt{improved\_long} 与 \texttt{improved\_longer} 等更深骨干使测试准确率最高达到 \textbf{81.60\%}（\texttt{exp7}）。\texttt{exp8\_improved\_longer\_12} 在 \texttt{exp7} 配方上继续加深至 12 个卷积层，测试仅 \textbf{69.50\%}，说明在本数据与训练预算下\textbf{并非越深越好}。\texttt{exp4}（SGD）与 \texttt{exp5}（较小 Adam lr）在固定 \texttt{exp2} 结构下未超过原 Adam 配置。

Grad-CAM 可视化进一步表明模型在多数正确样本上会关注物体区域，但错误样本仍存在背景干扰与细粒度混淆，说明模型已经学到一定语义线索，但距离完全稳健的图像理解仍有改进空间。后续可继续尝试更精细的数据增强策略、学习率调度搜索和更深层网络结构，并结合更强的可解释性方法做对照。
